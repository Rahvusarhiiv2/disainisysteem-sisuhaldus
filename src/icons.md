Rahvusarhiiv Design System uses [Google Material Symbols](https://fonts.google.com/icons) as the icon system which consists over 2,500 icons.

- Browse all the icons on the [Material Symbols page on Google Fonts](https://fonts.google.com/icons).
- [Check the Material Design Icons GitHub repository](https://github.com/google/material-design-icons) for the source files and different third-party npm packages for implementing it in your codebase.
- Check the official [Google Figma Plugin for the Material Symbols](https://www.figma.com/community/plugin/1088610476491668236) which helps designers easily search and add symbols in Figma.
