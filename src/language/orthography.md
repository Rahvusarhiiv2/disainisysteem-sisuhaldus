# Heakeelsus ja õigekeelsus

## Lühendikirjutus


**Elektronpost** või elektronkiri on korrektne lühendada {% good %}e-post{% endgood %}, {% good %}e-kiri{% endgood %}, mitte inglispäraselt {% bad %}*e-mail{% endbad %}, {% bad %}*email{% endbad %} või {% bad %}*mail{% endbad %}.

**Arvelduskonto** korrektne lühend on {% good %}ak{% endgood %}, mitte {% bad %}*a/a{% endbad %}. Lühend _ak_ sobib kasutamiseks arvetel, aga tekstis või vormis on soovitatav kasutada {% good %}konto nr{% endgood %} või {% good %}IBAN{% endgood %}.

**Registrikoodi** lühend on _rg-kood_, aga kuna see mõjub võõralt, võib Rahvusarhiivi tarkvaras lühendada registrikoodi {% good %}reg-kood{% endgood %} (oluline on, et lühendis ei oleks sidekriipsu asemel punkt).

Sobimatu on jätta teksti **euro** variandid {% bad %}*EURI{% endbad %}, {% bad %}*EUR-i{% endbad %} või {% bad %}*euri{% endbad %}. Lühend EUR sobib üksnes valuutatabelitesse. Tekstis, kus muust valuutast ei räägita, on sobiv sõna {% good %}euro{% endgood %} või tähis {% good %}€{% endgood %} (nt sidumata tekstis sh hinnakirjas, tabelipeas).

## Arvukirjutus

**Numbrid ühest kümneni kirjutatakse jooksvas tekstis sõnadega**, v.a statistikatekstides või muudes arvurikastes tekstides, samuti siis, kui vähemalt üks arv on komakohaga.

**Arvu ja mõõtühiku vahel on tühik** (nt {% good %}2 tk{% endgood %}, {% good %}3 lk{% endgood %}).

**Arvu ees asuvad miinus- ja plussmärk on tühikuta** (nt {% good %}-35{% endgood %}, {% good %}+1{% endgood %}).

**Kui tabel sisaldab ainult kuni neljakohalisi arve, siis numbreid ei rühmitata.** Seda tehakse alates viiekohalistest arvudest tagantpoolt kolmekaupa.

**Telefoninumbrite rühmitamist tuleb alustada tagantpoolt ja teha seda neljakaupa** (nt {% good %}744 1582{% endgood %}).

**Kümnendmurdudes on korrektne kasutada koma**, mitte punkti (nt {% good %}10,2 meetrit{% endgood %}).

**Vahemike märkimisel tuleb kasutada pikka kriipsu**, mitte sidekriipsu. Kriipsu ette ja taha pole vaja tühikuid lisada (nt {% good %}4000–6000{% endgood %}). Vahemikus kasutatavate näitajate ühikut ei ole vaja korrata (nt {% good %}2–4%{% endgood %}).

**Kellaaja kirjutamisel on õige panna tundide ja minutite vahele punkt**, mitte koolon (nt {% good %}10.15–10.30{% endgood %}). Kui ajavahemikus on üksnes täistunnid, ei ole nullide lisamine vajalik (nt {% good %}Oleme lahti E–R 9–19{% endgood %}), aga kui näiteks ajakava sisaldab ka muul ajal algavaid ettevõtmisi, on ühtsuse mõttes hea lisada igale poole täistunni nullid:

{% good %}10.00–12.00 Algus{% endgood %}
{% good %}12.00–12.45 Paus{% endgood %}
{% good %}12.45–13.45 Lõpp{% endgood %}

**Erandiks on tabelid, kus kuvatakse kuupäev koos kellaajaga.** Seal tuleb tundide ja minutite vahel kasutada koolonit.

**Üksinda asuvale araabia numbrile saab käändelõpu lisada alates ainsuse osastavast** (v.a 1–4, millel ei ole ei omastava ega osastava käände tunnust). Käändetunnus tuleb lisada **sidekriipsuga** (nt {% good %}11-lt{% endgood %}, {% good %}31-l{% endgood %}, {% good %}41-le{% endgood %}).

**Kui arv kuulub sõnaühendisse ja ühildub käändes, siis numbrile käändetunnust ei lisata** (nt {% good %}30 tellimusele{% endgood %}). Kui tegemist on aga pikema sõnajadaga ja käändetunnusega sõna jääb arvust kaugele, võib numbrit täiendada käändetunnusega (nt {% good %}30-le uurimissaali väljastatud tellimusele{% endgood %}).

**Kui omadussõna on välja kirjutatud, tuleb see arvuga kokku siduda sidekriipsu abil** (nt {% good %}25-liikmeline{% endgood %}, {% good %}125-leheküljeline{% endgood %}). Kui arvule järgnev ühik on aga lühendina esitatud, kirjutatakse see lahku (nt {% good %}120 a raamat{% endgood %}). Teistmoodi käitub protsendimärk, mis jääb arvuga alati kokku (nt {% good %}10% osakaal{% endgood %}).

**Tehtemärkide ette ja taha jäetakse tühik** (nt {% good %}2 + 3 = 5{% endgood %}). Tuleb jälgida, et kasutusel oleks korrektne korrutusmärk ({% bad %}*4 x 5{% endbad %} > {% good %}4 × 5{% endgood %}).

## Kirjavahemärgid

### Punkt

Kui **veebilingid ja meiliaadressid** jäävad lause lõppu, käib lause lõppu ikka punkt: {% good %}Lisateavet saab e-posti aadressilt eteatis@ra.ee. Vaata ka ra.ee.{% endgood %}

### Pikk kriips

Tänapäeva fondid sisaldavad mitmes pikkuses pikki kriipse (–, —), kuid nende eristamine ei ole õigekirja küsimus. **Rahvusarhiivi rakendustes tuleb pika kriipsuna kasutada enn-kriipsu (–).**

Pikka kriipsu kasutatakse arvu- ja ajavahemiku näitamiseks sõna kuni asemel, sealjuures ei lisata pika kriipsu ette ega taha tühikut.

### Hüüumärk

**Hüüumärk on omal kohal käskude, soovide ja palvete väljendamisel** (nt {% good %}Tähelepanu! Lisage fail!{% endgood %}). Väitlause lõpus mõjub see ülepingutatult (nt {% bad %}*Sisestatud turvakood ei ole õige!{% endbad %}).

### Jutumärgid

Eestis on kasutatud mitmesuguse kujuga jutumärke:

- saksapärased „…“ – sagedasimad 1920.- 1930. aastate trükistes ja eelistatud alates 2013. aasta ÕS-ist
- prantsuse ja vene «…» – levisid Nõukogude Liidu ajal
- inglise “…” – hakkasid levima 1990. aastatest koos arvutikirjaga
- soome ”…” – hakkasid levima 1990. aastatest alates
- „…” – valiti 2006. aastal ilmunud ÕS-i tarbeks, sarnanevad käsitsi kirjutatutega (kasutatakse ka ungari, poola, rumeenia, serbia keeles)

Standard soovitab, mitte ei käsi, ja kõige olulisem on kasutusühtlus. Rahvusarhiivi rakendustes on mõistlik kasutada **standardklaviatuuri ühepoolseid jutumärke "…".**

**Arvutikäsklused ja veebilehel esitatavad rubriigid, sakid, nupud** jms kirjutatakse jutumärkides, nt {% good %}Klõpsake nupul "Lisa". Tellimused kuvatakse jaotises "Kinnitamata tellimused".{% endgood %}

### Kaldkriips

**Murdudes** kirjutatakse kaldkriips **tühikuteta**: {% good %}2/15{% endgood %}.

**Jagamistehetes** jäetakse **mõlemale poole** kaldkriipsu **tühik**: {% good %}16 / 4 = 4{% endgood %}.

Kui arvulise näitaja juures on **ühikutest üks sõnaga välja kirjutatud, kaldkriipsu ei panda**, nt {% bad %}*15 tk/pakis{% endbad %} > {% good %}15 tk pakis{% endgood %}; {% bad %}*40 eurot/tund{% endbad %} > {% good %}40 eurot tund{% endgood %}.

Kui kaldkriips **eristab üksiksõnu, ei eraldata seda tühikutega**, nt {% good %}ja/või{% endgood %}, {% good %}linn/vald{% endgood %}, kui aga **kasvõi ühel pool on mitu sõna, tuleb mõlemale poole kaldkriipsu jätta tühik**, nt {% good %}olen rahul / ei ole rahul{% endgood %}, {% good %}ajavahemik / kuude arv{% endgood %}.

### Euro tähis
Euro tähis kirjutatakse summa järele (nt {% good %}34 €{% endgood %}).

## Ühildumine

**Veebiaadressi ei tohi lause lõpus jätta ühildamata** (nt {% bad %}*Täpsemaid juhiseid vaata ra.ee/vau{% endbad %}). Sellistes lausetes tuleb lisada lingitud aadressi ette sobivas käändes sõna ({% good %}Täpsemaid juhiseid vaata aadressilt ra.ee/vau{% endgood %}) või linkida veebitekstis sobivas käändes sõna ({% good %}Täpsemaid juhiseid vaata <u>virtuaalse uurimissaali</u> veebilehelt{% endgood %}).
