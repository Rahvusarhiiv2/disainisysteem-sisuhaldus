# Autotüpograafia

<p class="-mt-8 text-gray-300">Rahvusarhiivi rakendustes saab kasutada HTML-koodile automaatselt rakenduvat tüpograafiat</p>

Klass | Soovitus  
-- | --
prose | HTML-redaktoriga lisatud sisu jaoks

<article class="prose rounded bg-gray-50 my-10 px-6 py-4 w-full">
  <h2>
    Nooruse 3, 50411 Tartu
  </h2>
  <ul>
    <li>
      telefon 738 7521
    </li>
    <li>
      e-post tartu@ra.ee
    </li>
  </ul>
  <h3>
    Uurimissaal on avatud
  </h3>
  <ul>
    <li>
      esmaspäevast reedeni 9—19
    </li>
    <li>
      riiklike pühade eel 9—13
    </li>
  </ul>
  <h3>
    Säilikute väljastamine
  </h3>
  <ul>
    <li>
      Korraga võib uurimissaali välja tellida kuni 15 arhivaali.
    </li>
    <li>
      Tellitud arhivaalid jõuavad hoidlatest uurimissaali reeglina 1 tunni jooksul.
    </li>
  </ul>
</article>

```html
<article class="prose ..."></article>
```
